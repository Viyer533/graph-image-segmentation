How it works:

Image segmentation is the process of partitioning an image into multiple segments or regions, where each region corresponds to a separate object or part of the image. The program creates boundaries that satisfy global properties and specifies certain conditions for evaluating the evidence of a boundary between two sections. This app has been implemented using Python for the algorithm and Django framework for building the web app.

Instructions:
  • Click 'Choose File' to upload an image of your choice from your device.
  • Once uploaded, click 'Convert' to segment the given input image.
  • The segmented image will be displayed on the screen.
  • Click 'Download Image' to download the segmented image to your device.
